# pydigger-data

Data is collected by the [PyDigger backend](https://github.com/szabgab/pydigger.rs) and then displayed using the [PyDigger frontent](https://github.com/szabgab/pydigger-front).

To see the result visit the [PyDigger](https://pydigger.code-maven.com/) web site.

If you'd like to make improvements to this project open an issue or send a PR to either the backend or the frontend.
